Assignment 1, String Manipulation with Regular Expression

Due Date: 11:59pm, September 27, 2022

 

Purpose
  • Practice of string data manipulation in python
  • Practice of the use of regular expression in python

Description
When we search homolog sequences, we usually filter out low-complexity regions which have a repeated pattern of a small sequence segment. Suppose a low-complexity region is defined when the sequence segment of size between 2 and 5 is repeated consecutively at least 3 times. Write two python scripts: one uses a regular expression to find all low-complexity regions from an input DNA sequence, and the other finds all low-complexity regions without a regular expression. Your python scripts will take an input file name by standard input, read the input file of a DNA sequence in a FASTA format, and print the starting positions of all low-complexity regions to the screen by standard output. (The position of the first character of a sequence is 0.) Print the elapsed time of your two python scripts in microseconds to the screen.

Submission
Submit your two python files: "Assignment1-1.py" for the use of regular expression and "Assignment1-2.py" for the other.

Note
  • The input sequence is case-insensitive.
  • If the input file entered by a user does not exist, then print "No input file".
  • If the input has nothing, then print "No DNA sequence".
  • If the input is not a DNA sequence, then print "No DNA sequence". The input sequence can be validated using a regular expression in both python scripts.
  • If the input file does not follow the FASTA format, then print "No correct format".
  • If the input file has multiple sequences, then read the first sequence.
  • If the input sequence does not include any low-complexity region, then print "No low-complexity region found".
  • If two low-complexity regions are overlapping, then return the low-complexity region occurring at first, and search more low-complexity regions from the remaining part of the sequence.
